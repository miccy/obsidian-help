Plugin **Daily notes** (Denní poznámky) je základní plugin, který vám umožňuje snadno vytvářet a přistupovat k denním záznamům deníku.

## Použití

- **Vytvořit dnešní denní poznámku**: Klikněte na ikonu **Open today's daily note** (Otevřít dnešní denní poznámku) na pásu karet (ikona kalendáře).
- **Přejít na předchozí/další denní poznámku**: Můžete použít příkazy palety příkazů "Open next daily note" (Otevřít další denní poznámku) a "Open previous daily note" (Otevřít předchozí denní poznámku) (je užitečné k nim přiřadit klávesové zkratky).

## Nastavení

Plugin denních poznámek můžete přizpůsobit v **Settings** (Nastavení) -> **Daily notes**.

- **Date format** (Formát data): Formát použitý pro název souboru (např. `RRRR-MM-DD`).
- **New file location** (Umístění nového souboru): Složka, kde se vytvářejí denní poznámky.
- **Template file location** (Umístění souboru šablony): Šablona, která se má použít pro nové denní poznámky.
- **Open daily note on startup** (Otevřít denní poznámku při spuštění): Automaticky otevře dnešní denní poznámku při spuštění Obsidianu.
