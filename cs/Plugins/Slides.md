Plugin **Slides** (Prezentace) vám umožňuje prezentovat vaše poznámky jako slideshow.

## Vytváření snímků

Pro vytvoření sady snímků použijte `---` k oddělení jednotlivých snímků.

```markdown
# Snímek 1

Toto je první snímek.

---

# Snímek 2

Toto je druhý snímek.
```

## Spuštění prezentace

1. Otevřete poznámku, kterou chcete prezentovat.
2. Otevřete paletu příkazů.
3. Vyhledejte "Slides: Start presentation" (Prezentace: Spustit prezentaci).

Tím se otevře nové okno s vaší prezentací.

## Ovládání

- **Další snímek**: Šipka vpravo, Mezerník
- **Předchozí snímek**: Šipka vlevo
