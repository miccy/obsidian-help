Plugin **Unique note creator** (Tvůrce jedinečných poznámek), dříve známý jako Zettelkasten prefixer, vám pomáhá vytvářet poznámky s jedinečnými identifikátory (ID) na základě aktuálního data a času.

## Použití

- **Pás karet (Ribbon)**: Klikněte na ikonu **Create new unique note** (Vytvořit novou jedinečnou poznámku).
- **Paleta příkazů**: "Unique note creator: Create new unique note".

Tím se vytvoří nová poznámka s časovým razítkem jako názvem (např. `202310271430`).

## Nastavení

Formát jedinečného ID můžete přizpůsobit v **Settings** (Nastavení) -> **Unique note creator**.

- **New file location** (Umístění nového souboru): Kde se ukládají nové jedinečné poznámky.
- **Template file location** (Umístění souboru šablony): Šablona, která se má použít pro nové jedinečné poznámky.
- **Unique prefix format** (Formát jedinečného prefixu): Formát data/času pro ID (výchozí: `RRRRMMDDHHmm`).
