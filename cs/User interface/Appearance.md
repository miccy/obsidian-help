Vzhled a chování Obsidianu si můžete přizpůsobit v **Settings** (Nastavení) -> **Appearance** (Vzhled).

## Základní motiv

Vyberte si mezi režimem **Light** (Světlý) a **Dark** (Tmavý), nebo nechte Obsidian přizpůsobit se nastavení vašeho systému.

## Barva zvýraznění (Accent color)

Vyberte si vlastní barvu zvýraznění pro odkazy, tlačítka a další zvýrazněné prvky.

## Motivy (Themes)

Obsidian podporuje komunitní motivy, které mohou zcela změnit vzhled aplikace.

1. Klikněte na **Manage** (Spravovat) vedle **Themes**.
2. Procházejte dostupné motivy.
3. Klikněte na **Install and use** (Nainstalovat a použít) pro aplikaci motivu.

## Písmo (Font)

Můžete si přizpůsobit písmo použité v rozhraní a editoru.

- **Interface font** (Písmo rozhraní): Používá se pro nabídky, postranní panely a průzkumníka souborů.
- **Text font** (Písmo textu): Používá se pro editor a režim čtení.
- **Monospace font** (Neproporcionální písmo): Používá se pro bloky kódu a frontmatter.

## CSS snippety

Pro detailnější kontrolu můžete použít CSS snippety (úryvky) k aplikaci vlastních stylů nad rámec vašeho motivu.

1. Kliknutím na ikonu složky vedle **CSS snippets** otevřete složku snippetů.
2. Vytvořte v této složce soubor `.css` a přidejte svůj CSS kód.
3. Obnovte seznam v Obsidianu a snippet zapněte.
