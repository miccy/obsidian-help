Obsidian přichází s širokou škálou klávesových zkratek, které urychlí váš pracovní postup při úpravách.

Všechny zkratky si můžete prohlédnout a přizpůsobit v **Settings** (Nastavení) -> **Hotkeys** (Klávesové zkratky).

## Běžné klávesové zkratky pro úpravy

| Akce | Windows/Linux | macOS |
| :--- | :--- | :--- |
| **Tučné** | `Ctrl + B` | `Cmd + B` |
| **Kurzíva** | `Ctrl + I` | `Cmd + I` |
| **Přeškrtnuté** | `Ctrl + S` | `Cmd + S` |
| **Zvýraznění** | `Ctrl + =` | `Cmd + =` |
| **Vložit odkaz** | `Ctrl + K` | `Cmd + K` |
| **Zpět** (Undo) | `Ctrl + Z` | `Cmd + Z` |
| **Znovu** (Redo) | `Ctrl + Y` | `Cmd + Shift + Z` |
| **Přepnout sbalení** | `Ctrl + '` | `Cmd + '` |
| **Odsadit** | `Tab` | `Tab` |
| **Zrušit odsazení** | `Shift + Tab` | `Shift + Tab` |

## Navigační klávesové zkratky

| Akce | Windows/Linux | macOS |
| :--- | :--- | :--- |
| **Přejít na definici** | `F12` | `F12` |
| **Navigovat zpět** | `Alt + Šipka vlevo` | `Cmd + Opt + Šipka vlevo` |
| **Navigovat vpřed** | `Alt + Šipka vpravo` | `Cmd + Opt + Šipka vpravo` |
| **Otevřít paletu příkazů** | `Ctrl + P` | `Cmd + P` |
| **Rychlý přepínač** | `Ctrl + O` | `Cmd + O` |

## Přizpůsobení zkratek

Můžete změnit kteroukoli z výchozích zkratek nebo přiřadit nové k příkazům, které je nemají.

1. Otevřete **[[Settings|Nastavení]]**.
2. Vyberte **Hotkeys** (Klávesové zkratky).
3. Pomocí vyhledávacího panelu najděte příkaz, který chcete přizpůsobit.
4. Klikněte na tlačítko **Customize this command** (ikona klávesnice).
5. Stiskněte kombinaci kláves, kterou chcete použít.
